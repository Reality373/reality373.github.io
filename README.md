### 📶 Wireless Penetration Testing Toolkit
`C++` `ESP32` `Arduino IDE` `802.11` `Wireshark` `PCAP` `Bluetooth/BLE`

A portable, hardware-based wireless security auditing tool built on the ESP32 microcontroller. Designed for field security assessments of 802.11 infrastructure.

- Active/passive packet sniffing and targeted deauthentication attacks
- EAPOL/PMKID frame capture for WPA2 handshake analysis
- Beacon spamming to evaluate infrastructure resilience
- Bluetooth/BLE sniffing for rogue device detection
- Logs directly to Wireshark-compatible PCAP files

### 🔌 IoT-Based Plug-and-Play Vulnerability Scanner
`C++` `Arduino Pro Micro` `ESP8266` `Exploit-DB` `Windows Registry`

A hardware vulnerability scanner that auto-executes security audits the moment it's plugged into a Windows host — no software installation required.

- Evaluates system settings, registry configs, and installed software versions
- Cross-references findings against Exploit-DB for known CVEs
- Generates detailed reports with risk scores and threat severity categorization (Low / Medium / High)
